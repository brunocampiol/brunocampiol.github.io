Home

This website is intended to share and showcase some of the things I have done. I hope you enjoy it. Initially, my site was built using PHP, as it was the only language available for free hosting. Next, I moved it to .NET Core, as I had a subscription that allowed me to run something on Azure. Finally, I believe this is the final stage, as GitHub Pages is free to use, and any backend coding can (and should) be done elsewhere.

You can explore my content here. Some JavaScript apps have already been ported over, such as the JS Pi Calculator and Box2D JS, which can be found on the Projects page. Also, be sure to check out some new gaming-related posts.
