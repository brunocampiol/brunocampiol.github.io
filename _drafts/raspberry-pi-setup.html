Raspberry Pi 2 model B self hosting

Goal

The goal was to give this hardware a retry as a (cheap) self hosting device.

Hit: You can have it running virtually free by using your router usb port.

It is required to have a DNS domain name.

Install OS

Just follow official steps: Install Raspberry Pi OS using Raspberry Pi Imager -> Raspberry Pi OS

Remember the raspi-config tool. It has some good options to change some settings in the raspberry pi board.
